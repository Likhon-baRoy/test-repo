* Quick Reference: Time Complexity ($O$)
A "cheat sheet" for quick estimations during contests.

| Complexity | Name | Common Examples |
| :--- | :--- | :--- |
| $O(1)$ | Constant | Accessing array index, HashMap =get= |
| $O(\log N)$ | Logarithmic | Binary Search, Heap operations |
| $O(N)$ | Linear | Kadane's, Linear Search, LL traversal |
| $O(N \log N)$ | Linearithmic | Merge Sort, Quick Sort, Map/Set operations |
| $O(N^2)$ | Quadratic | Nested loops (Bubble sort, matrix traversal) |
| $O(2^N)$ | Exponential | Recursive subsets, Brute-force DFS |

* Essential Math Tricks
Keep these "Mental Shortcuts" here for fast retrieval.

- *Subarray Count:* A string/array of length $N$ has $N(N+1)/2$ non-empty subarrays.
- *Subsequence Count:* A sequence of length $N$ has $2^N - 1$ non-empty subsequences.
- *Bitwise Power:* $2^N$ is the same as =(1 << N)=.
- *Parity Check:* =(x & 1)= is faster than =(x % 2 != 0)=.
